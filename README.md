# pktxdiag.sty

## A TikZ-based LaTeX package for drawing packet exchange diagrams

Have a look at the wiki for a few examples and the user guide in pdf
form.

The user guide source is `pktxdiag.tex`, compile it to pdf with

    pdflatex pktxdiag
    pdflatex pktxdiag
    makeindex pktxdiag
    pdflatex pktxdiag

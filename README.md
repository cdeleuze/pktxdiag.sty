# pktxdiag.sty

## A TikZ-based LaTeX package for drawing packet exchange diagrams

Have a look at the wiki for a few examples and the userguide in pdf
form.

The userguide source is `userguide.tex`, compile it to pdf with

    pdflatex userguide
    pdflatex userguide
    makeindex userguide
    pdflatex userguide
